A modern web application built for EcoCiel, a solar energy company based in Fez, Morocco.

## 🚀 Project Overview

This application helps EcoCiel generate leads and showcase their solar energy services to potential clients.

Users can:

* 📊 Estimate their solar energy savings
* 📩 Submit a request for a free quote
* 📞 Contact easily the company

---

## ⚙️ Tech Stack

* React.js
* Next.js
* Tailwind CSS
* Supabase (Database)
* Vercel (Deployment)

---

## 🔥 Features

* Responsive landing page
* Solar savings calculator
* Contact form with database integration
* WhatsApp quick contact
* Clean and modern UI

## 🧪 Local Setup

```bash
git clone https://github.com/medfassi20/ecociel-energie-application.git
cd ecociel-energie-application
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the root:

```
VITE_SUPABASE_URL=your-url
VITE_SUPABASE_ANON_KEY=your-key
```
